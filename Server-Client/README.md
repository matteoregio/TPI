 Quiz Server-Client
 ---


Questo progetto si basa sull'architettura Client-Server, il server quando si collega al client invia la prima domanda, e aspetta la risposta da parte del client. Dopodichè il server verifica se la risposta è corretta o meno, e invia la risposta al client, insieme alla domanda successiva. All'ottava domanda il quiz finisce.

Come argomento per le domande abbiamo scelto la Formula1.
